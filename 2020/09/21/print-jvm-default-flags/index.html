打印JVM默认参数

2020-09-21 :: Zhige
#JVM 

运行java程序时加入-XX:+PrintFlagsFinal可以打印出JVM启动时所设置的所有的默认flags的值！

本人的JDK的版本情况：

1$ java -version
2java version "1.8.0_151"
3Java(TM) SE Runtime Environment (build 1.8.0_151-b12)
4Java HotSpot(TM) 64-Bit Server VM (build 25.151-b12, mixed mode)
5

我们假设启动参数：

1java  -XX:+PrintFlagsFinal messageSenderTest.JVMTest2

这样就可以打印出JVM所有flags的默认值啦！
